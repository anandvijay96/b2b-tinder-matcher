# NMQ B2B Match

A Tinder-style B2B matching mobile app that connects verified businesses using AI-driven intent matching.

## Tech Stack

- **Expo** SDK 54 (React Native 0.81+)
- **Expo Router** v6 (file-based routing)
- **NativeWind** v4 (Tailwind CSS for React Native)
- **Zustand** 5 (state management)
- **TypeScript** 5.9+ (strict mode)

## Getting Started

```bash
# Install dependencies
bun install

# Start development server
bun start

# Run on Android
bun android

# Run on iOS
bun ios
```
